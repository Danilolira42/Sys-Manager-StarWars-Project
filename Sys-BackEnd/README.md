Para rodar o projeto em local host você deve:

Instalar o maven na versão: Apache Maven 3.9.14

Configurar as variáveis de ambiente em seu computador para que acesse o .bin do Maven

Baixar o Temurin com o Java na versão 21 para utilizar o framework do Java Quarkus.

Running:

Para rodar o projeto Back-End, clique com o botão direito sobre a pasta Back-End -> Abrir no Terminal Integrado e execute o comando abaixo:

mvn quarkus:dev "-Dquarkus.enforceBuildGoal=false"
