
Para rodar o projeto de Front-End se certifique de ter instalado em sua máquina a versão mais recente do node.js para poder usar o npm install e gerar a pasta de node_module com os bundlers necessários para rodar o projeto e siga os passos abaixo:

Dependências:
npm install react-router-dom
npm install react-icons


Clique com o botão direito do mouse sobre -> Abrir em Terminal Integrado e rode o seguinte comando npm:

npm run dev

Já está configurado para abrir automaticamente no navegador a aplicação, não é necessário clicar sobre o link.
